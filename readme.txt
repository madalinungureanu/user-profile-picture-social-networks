Allow users to add social networks to their profile.

== Description ==

Allow users to add/edit/display their social networks. Requires <a href="https://wordpress.org/plugins/metronet-profile-picture/">User Profile Picture</a> version 2.3.10 and up.

There's also a Gutenberg block and a template tag for outputting your social networks. <a href="https://mediaron.com/user-profile-picture/social-networks/">View the documentation</a>.

https://www.youtube.com/watch?v=RETBoLQS_-o

== Installation ==

1. Requires User Profile Picture 2.3.10 and up.
2. Requires WordPress 5.0.
3. Requires FontAwesome 5.0.
4. Navigate to your user profile and begin adding social networks for you and your users.
5. Use a template function or a Gutenberg block to output the social networks.

<a href="https://mediaron.com/user-profile-picture/social-networks/">View the documentation</a>.
